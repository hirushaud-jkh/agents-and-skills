# Templates Directory

Place your company `.pptx` template files here. The deck skill auto-detects any `.pptx` files in this directory.

## Setup

1. Copy your branded PowerPoint template(s) into this folder
2. Run `/deck` — the skill will list available templates and let you pick one
3. On first use with a new template, the skill inspects its layouts and placeholders automatically

## Notes

- Templates are `.gitignore`d — they stay local and are not committed to the repo
- Each user/team can have different templates without conflicts
- The skill works with any `.pptx` file — no special naming convention required
